# ArtConnect 
**Interactive Digital Art Gallery Platform**

---

##  Project Overview

**ArtConnect** is a client-side web application designed to showcase artworks and artist portfolios in an engaging digital environment. The platform allows users to create artist profiles, upload artworks using image URLs, browse a categorized gallery, and interact through likes and comments.

This project is developed as part of the **FEWDL (Front-End Web Development Lab)** external assessment and focuses on usability, responsive design, and core front-end concepts using **HTML, CSS, and JavaScript**.

---

##  Objectives

- Provide a simple platform for managing artists and artworks
- Display artworks in a visually appealing gallery
- Enable searching and filtering by category
- Allow user interaction through likes and comments
- Demonstrate front-end development skills without backend dependency

---

##  Features / Functional Requirements

###  Core Features
- Artist profile creation
- Artwork upload using **direct image URLs**
- Categorized gallery display
- Search artworks by title
- Filter artworks by category
- Artwork details view using a **lightbox**
- Responsive layout for desktop and mobile devices

###  Optional Enhancements Implemented
- Like system for artworks
- Comment system for user interaction
- Lightbox preview with artwork details
- Clean and aesthetic UI using CSS variables
- Sample data for demonstration

---

##  Technologies Used

- **HTML5** – Structure and layout
- **CSS3** – Styling, layout, responsiveness
- **JavaScript (Vanilla JS)** – Application logic and interactivity

> This project is **fully client-side** and does not use any backend or database.

##  How to Run the Project

1. Download or clone the project folder
2. Open `index.html` in any modern web browser (Chrome recommended)
3. No server setup or installation required

---

##  Sample Data

The project includes predefined:
- Artist names
- Artwork entries with valid image URLs (Unsplash)
- Categories and tags

This allows immediate demonstration without manual input.

---

##  Limitations

- No backend or database (data resets on page refresh)
- Image upload via URL only (no file upload)
- No authentication system
